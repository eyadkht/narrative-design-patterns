Pattern 06 Flow Emotion

Gradual Reveal

Introduce complexity layer by layer so each insight lands before the next one arrives.

Why it works

Dumping all information at once overwhelms readers before they have built a mental model. Staged disclosure creates a narrative arc, each layer adds meaning to the one before, and each finding lands with its full emotional weight.

How to use it

Use a "Next" button or scroll trigger to add layers: start with bare axes, then data points, then a trend line, then an annotation, then the conclusion. Animate each arrival so the change is impossible to miss.

Real-world example
