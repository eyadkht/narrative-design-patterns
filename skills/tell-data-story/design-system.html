05

SVG chart conventions

All charts are inline SVG, no libraries. Thin 1.5px ink strokes, one category color per series, mono labels for axes and callouts.

SVG rules of thumb

stroke-width: 1.5      // axes and chart outlines
stroke-width: 1        // grid, baselines
stroke-width: 2        // primary series (lines)
fill: var(--cat-X)     // one category color per series
font-family: ui-monospace, font-size: 8–10px   // axis ticks & callouts
stroke-dasharray: 2 2  // annotation leader lines

09

Motion

Three durations, one easing curve. --dur-slow for narrative reveals, --dur for default state changes, --dur-fast for hover feedback. Respects prefers-reduced-motion.

--dur-fast · 120ms

Hover lifts, button presses, micro-feedback.

--dur · 240ms

Default. Filter changes, chip activation.

--dur-slow · 600ms

Narrative reveals, gradual-reveal stages.

Easing · cubic-bezier(.2,.6,.2,1)

SVG animation snippets

/* Layered reveal (gradual-reveal) */
.layer { opacity: 0; transition: opacity var(--dur-slow) var(--ease); }
.layer.is-on { opacity: 1; }

/* Bar grow-in */
rect.bar { transform-origin: bottom; transform: scaleY(0);
           transition: transform var(--dur-slow) var(--ease); }
rect.bar.is-on { transform: scaleY(1); }

/* Path draw-on */
path.line { stroke-dasharray: 600; stroke-dashoffset: 600;
            transition: stroke-dashoffset var(--dur-slow) var(--ease); }
path.line.is-on { stroke-dashoffset: 0; }

/* Counter-up */
const t0 = performance.now();
requestAnimationFrame(function tick(t) {
  const k = Math.min(1, (t - t0) / 600);
  el.textContent = Math.round(target * (0.2 + 0.8 * k));
  if (k < 1) requestAnimationFrame(tick);
});
10

Accessibility

Rules every generated demo has to keep.

  • · Color is never alone, every category color is paired with a label.
  • · Yellow is background only, never foreground text or thin strokes on white.
  • · Focus ring, 2px ink ring on every interactive element.
  • · Min target, 36×36px on touch.
  • · prefers-reduced-motion, all --dur-* tokens drop to 0ms automatically.
  • · SVG aria-hidden on decorative charts, <title> on data charts describing the message.
